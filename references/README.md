# References

Official World Bank design documentation and extracted references.

## Documents

| File | Description |
|------|-------------|
| [`WBG_BrandGuidelines.pdf`](WBG_BrandGuidelines.pdf) | Official WBG Brand Guidelines — comprehensive brand identity standards including logo usage rules, color system, typography, and do/don'ts |
| [`WBG-Digital-Look-StyleguideV2.pdf`](WBG-Digital-Look-StyleguideV2.pdf) | WBG Digital Look & Feel Style Guide V2 — covers PowerPoint templates, social media, brushstroke/graphic elements, vertical color system, and Canva templates |
| [`WBG_Banner_Templates.pdf`](WBG_Banner_Templates.pdf) | WBG Banner Templates — backdrop and event signage templates |
| [`styleguide.md`](styleguide.md) | Extracted and edited text version of the digital style guide — color palettes with full tint tables, typography, layout principles, and a prompt engineering template for AI workflows |

## Notes

- The WBG style guide is primarily designed for **PowerPoint and social media** assets, not data visualization. Color codes and typography translate well to front-end development, but the vertical palette system does not work well for multi-category chart data.
- For data visualization color guidance, see [`../data-viz/palettes.md`](../data-viz/palettes.md).
- For the adapted front-end design system, see [`../design-guide/`](../design-guide/).
