## WB Logos (`wb-logos/`)

Official WBG logo assets organized by layout and color variant. Source: WBG Brand Guidelines.

| Subfolder | Description |
|-----------|-------------|
| `vertical/` | Globe + "THE WORLD BANK" stacked layout |
| `horizontal/` | Globe + "THE WORLD BANK" side-by-side layout |
| `globe/` | Globe mark only, no text |

Each has `color/`, `white/`, `black/`, and `grayscale/` variants. White folders include both gradient (preferred) and solid versions. Each variant ships in 3 resolutions: `-web`, standard, and `-high`.

See [`wb-logos/README.md`](wb-logos/README.md) for usage guidance.

## OCHA Icon Sets

Three separate OCHA releases are kept here as distinct folders:

- **`ocha-box-icons/`** — Activity-focused icons, each available in 32px, 64px, and 100px sizes, with standard and blue-box style variants. Naming pattern: `activity_[name]_[size]px_icon[_bluebox].png`
- **`ocha-white-svg/`** — Full OCHA humanitarian icon library in white SVG format (387 icons)
- **`ocha-humanitarian-icons-16/`** — OCHA humanitarian icon set v16, available as both SVG and PNG. Naming pattern: `noun-[concept]-[number].[ext]`

> **Note:** Organization of OCHA icon sets may be revisited in the future.
