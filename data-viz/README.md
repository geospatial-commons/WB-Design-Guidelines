# Data Visualization

Color palettes and aesthetic guidelines for charts, graphs, and maps in WB data products.

> **Note:** Standard WB brand colors are not optimized for data visualization. The palettes in this section are developed to be on-brand but work well for charts and maps — legible, accessible, and visually balanced.
